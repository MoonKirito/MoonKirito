# MoonKirito Profile README Setup

把整个 `MoonKirito` 文件夹上传到 GitHub 中与你用户名同名的公开仓库：`MoonKirito/MoonKirito`。

推荐命令：

```bash
cd MoonKirito
git init
git add .
git commit -m "chore: initialize profile README"
git branch -M main
git remote add origin https://github.com/MoonKirito/MoonKirito.git
git push -u origin main
```

注意：仓库必须是 Public，并且仓库名必须和 GitHub 用户名完全一致，主页 README 才会显示在个人主页。
